# A-Share Trading System (A股交易系统) 使用指南

欢迎使用 A-Share Trading System (ashare-trader)！
本系统基于“趋势跟随 + ATR 波动率风控”的专业交易理念设计，旨在帮助投资者在 A 股市场实现稳定增长，规避大幅回撤。

本指南将帮助您了解如何使用该 Skill 进行大盘择时和个股分析。

---

## 核心功能 (3层架构)

### Layer 1: 宏观择时 (Macro Timing)
**目的**：判断大盘环境是否适合交易。
- **监控对象**：上证指数、深证成指、创业板指。
- **信号输出**：
  - 🔴 **RED (红灯)**：空头市场或下跌趋势 -> **停止交易 / 空仓**。
  - 🟡 **YELLOW (黄灯)**：震荡市或指数分化 -> **谨慎操作 / 防御为主**。
  - 🟢 **GREEN (绿灯)**：多头市场或上涨趋势 -> **积极交易 / 进攻**。

### Layer 2: 市场情绪 (Market Sentiment)
**目的**：多维度感知市场“水温”，避免单一指标失真。
- **监控因子**：
  1.  **北向资金 (Smart Money)**：外资流向，判断聪明钱态度。
  2.  **指数强度 (Index Performance)**：三大指数的平均涨跌幅。
  3.  **量能分析 (Volume)**：今日成交量与 5日均量对比，判断是放量还是缩量。
  4.  **市场广度 (Sampled Breadth)**：通过采样 20 只行业龙头股的涨跌情况，判断市场整体赚钱效应。
  5.  **领涨板块 (Leading Sectors)**：监控核心行业 ETF（证券、半导体、新能源、医药、消费、银行、军工、白酒）的实时表现，捕捉市场热点。
  6.  **技术指标 (RSI)**：计算上证指数的 6日 RSI，判断大盘是否超买（过热）或超卖（反弹机会）。
- **信号输出**：
  - **情绪评分 (0-100)**：综合上述四因子打分。
  - **Mood**：
    - > 80: 极度贪婪/过热 (Extremely Greedy)
    - 65-80: 贪婪/乐观 (Greedy)
    - 35-65: 中性 (Neutral)
    - 20-35: 恐惧/悲观 (Fearful)
    - < 20: 极度恐慌 (Extremely Fearful)

### Layer 3: 个股策略与风控 (Individual Strategy)
**目的**：在确认趋势的前提下，计算安全的买入点和仓位。
- **系统集成**：自动根据 Layer 1 (宏观) 和 Layer 2 (情绪) 调整仓位。
  - 🔴 大盘红灯 -> 0% 仓位 (禁止开新仓)。
  - 🟡 大盘黄灯 -> 50% 仓位 (防御性减半)。
  - 😨 情绪极度恐慌/过热 -> 80% 仓位。
- **趋势判断**：
  - **买入条件**：收盘价 > MA20 > MA60 (均线多头排列)。
  - **卖出/观望**：不满足上述条件。
- **风险控制 (ATR)**：
  - **止损价**：当前价格 - 2倍 ATR (平均真实波幅)。
  - **仓位管理**：确保单笔交易止损带来的损失不超过总资金的 **1%**。

## 快速开始

### 1. 环境准备
确保您已安装 Python 环境及所需依赖：
```bash
pip install -r requirements.txt
```

### 2. 常用命令

#### 场景一：大盘体检 (Market Check)
**想知道今天能不能买股票？先看大盘脸色。**

```bash
python src/main.py --action market_check
```
**输出解读**：关注最后的 `MARKET SIGNAL`。如果是红灯，建议休息；如果是绿灯，可以选股。

#### 场景二：个股分析 (Analyze Stock)
**看好某只股票（例如茅台），想知道具体的买点和仓位。**

参数说明：
- `--symbol`: 股票代码 (如 `600519`, `000001`)。
- `--capital`: 您的总账户资金 (用于计算风控仓位，如 `500000`)。

**示例 1：分析贵州茅台 (600519)，资金 50 万**
```bash
python src/main.py --action analyze_stock --symbol 600519 --capital 500000
```

**示例 2：分析平安银行 (000001)，资金 10 万**
```bash
python src/main.py --action analyze_stock --symbol 000001 --capital 100000
```

## 结果解读指南

当您运行个股分析时，系统会输出如下关键信息：

1.  **Trend (趋势)**：
    - `UPTREND (Buy Zone)`：**可以买入**。
    - `SIDEWAYS` 或 `DOWNTREND`：**观望/卖出**。

2.  **Volatility (ATR)**：
    - 股票的波动幅度。ATR 越大，波动越剧烈，系统建议的仓位会越小（为了安全）。

3.  **Stop Loss Price (止损价)**：
    - 这里的价格是您的“生命线”。如果买入后价格跌破此价位，**必须无条件止损**。

4.  **Suggested Position (建议仓位)**：
    - 系统根据您的总资金和风险偏好（1%），自动计算出的**最大安全持仓股数**。
    - **切记**：不要超过这个仓位，这是风控的核心！

5.  **STRATEGY SIGNAL (策略信号)**：
    - `BUY VALID`：趋势完美，资金管理通过，**建议买入**。
    - `WAIT / NO TRADE`：条件不满足，**建议空仓或等待**。

## 常见问题

**Q: 为什么系统建议我买 0 股？**
A: 这通常是因为：
1. 目前大盘环境恶劣（Layer 1 红灯），系统强制风控。
2. 目前个股趋势不是上涨趋势（MA20 < MA60 或 价格 < MA20）。
3. 或者波动率太高，导致止损空间过大，为了保护本金，系统建议不交易。

**Q: 我可以修改风险比例吗？**
A: 目前默认是 1% 的激进风控。如果需要修改，可以在代码 `src/main.py` 中调整 `risk_per_trade` 参数。

**Q: 支持哪些股票？**
A: 支持所有沪深 A 股。请输入 6 位数字代码（如 `600519`）。
